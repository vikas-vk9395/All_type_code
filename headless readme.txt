
To make Computer headless
open terminal
enter following command	
	sudo mv /usr/share/X11/xorg.conf.d/xorg.confx /usr/share/X11/xorg.conf.d/xorg.conf
Then restart the computer.

To make Computer with monitor
open terminal
enter following command
	sudo mv /usr/share/X11/xorg.conf.d/xorg.conf /usr/share/X11/xorg.conf.d/xorg.confx 
Then restart the computer.
